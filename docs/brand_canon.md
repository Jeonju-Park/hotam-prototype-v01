# 브랜드 캐논 — 호탐(HOTAM) 확정본 (2026-07)

> 브랜딩은 **확정 완료** 상태다. 구 프로젝트 가이드(01)의 "브랜드명·컬러 미정" 문구는 폐기. 이 문서가 브랜드의 단일 진실.

## 1. 이름
- **호탐 / hotam** — '호시탐탐(虎視眈眈)'에서. 좋은 것을 노려보다 확실할 때만 낚아채는 태도 = 별점 대신 직접 비교로만 기록하는 제품 철학.
- 4중 의미: 虎(호랑이) · 好(좋음 — 좋은 것만 남긴다) · 探(탐색) · 貪(식탐 — 마스코트의 성격)
- 표기 분담: **디지털·글로벌·법적 = hotam(소문자)** / **오프라인·세로 간판·국내 캠페인 = 호탐**
- ⚠️ KIPRIS 상표('호탐'/'HOTAM')·스토어 동일명 확인 미완 — 착수 전 필수.

## 2. 마스코트 — 탐이 (식탐 가득한 골목 호랑이)
- 서사: '해와 달이 된 오누이' 호랑이의 후예. 밤 골목의 불 켜진 간판 앞에서 코를 킁킁대는. "맛집 하나 알려주면 안 잡아먹지."
- 불변 6요소: ①이마 약식 줄무늬 3획(서명) ②둥근 삼각 귀 ③뺨 지그재그 수염털 ④통통한 발 ⑤플랫 솔리드(잉크 라인) ⑥2등신·이목구비 하단 배치
- 캐논 포즈: 간판 윗변을 잡고 빼꼼(=호시탐탐). 로고 문맥에선 이 포즈만.
- 노출 상한: **한 화면에 탐이 1회.** 화면의 주인공은 사용자의 기록.
- 금지: 파란 몸통·짝눈·갓 쓴 까치(케데헌 더피 식별 요소), 포식 묘사(위협은 항상 허풍으로 끝남).

## 3. 브랜드 보이스 — 따뜻한 장난기, "안 잡아먹지" 화법
- DO: 위협인 척하는 애교 / 행동을 짧고 구체적으로("3초면 끝나") / 수집·자랑 욕구 짚기 / 탐이 감각어("킁킁")
- DON'T: CS 말투("고객님의 소중한…") / 영어 남발 / 협박·공포 소구 / 이모지 도배 / 허세 추상어("미식의 진정성을 큐레이션")
- **배분 원칙:** 인풋·라벨·버튼 = 담백(관용구형, 예: 검색 placeholder "식당, 사람 검색" 또는 문장형 "오늘은 어느 집을 노려볼까요?") / 위트 = 빈 상태·에러·온보딩에 배분. 검색 placeholder는 로테이션 운영(브랜드형·스코프형·공감형 순환)으로 문구별 전환 데이터 수집.

## 4. 로고 (기획 문서에서 참조할 수준만)
- 구조: **간판형 시그니처** — 심볼(간판+빼꼼 탐이) + 로고타입. 반응형 락업 6종(풀/심볼/가로/워드마크 국·영문).
- 로고타입: 소문자 `hotam`, 클린 지오메트릭 + 장치 1개. **B안 'signboard t' 확정 방향** — 훅리스 직선 스템, 어센더 8° 컷, 크로스바 좌 연장(6/8/10% 게이트 중)·우 축소(좌:우 1.6~1.8:1). 장치는 형태로만, 색 분리 금지.
- 앱 아이콘 = 심볼 단독(텍스트 없음). 상세 스펙은 브랜드 문서(brand_hotam_logotype_guide) — 기획 폴더 범위 밖.

## 5. 무드 키워드 (와이어프레임·카피 톤 판단용)
정직한 비교 · 식욕을 돋우는 · 따뜻한 장난기 · 레트로 골목 라이트박스(요시노야·을지로) · 수집과 자랑. 색 총량: 오렌지 화면당 <10%, forest+cocoa <10%, status 상시 노출 금지.
