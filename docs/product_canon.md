- NSM: 주간 신뢰 기록 수. Kill Criteria: 숫자 **TBD**(권장 출발선 D7≥25%·D30≥12%·납득률≥60% — 출시 전 확정·고정 필수)

## 3. 스코어링 엔진 (전 문서가 지켜야 할 불변 규칙)
- 내부 Elo ↔ 표시점수(10점, 소수1) 단조 변환. **점수 직접 수정 경로 없음** — 교정은 리스트 드래그·"점수 이상해요" → 전부 비교 이벤트로 환원.
- 밴드: 좋음 7.0~10 / 그저 4.0~6.9 / 별로 0~3.9 `[제안값 — O8 튜닝 대상]`. **비교는 같은 밴드 안에서만.**
- 콜드스타트: 신규 기록=구간 중앙값, 신규 식당=카테고리·가격 prior → K감쇠. 신뢰도 비례 정밀도(RD 낮으면 소수점, 얇으면 "8점대").
- 큐레이션 점수: 본인 데이터 기반 예측, 데이터 얇으면 전체 평점으로 **정직 전환**.

## 4. 소셜 원칙
비대칭 팔로우 · **자동 친구추가·강제초대 영구 금지** · 친구 0명 → 전체 평판으로 graceful degrade · S7 댓글·공유 풀세트는 단계화 검토(**팀 결정 대기**).

## 5. Out of Scope (MVP)
협업필터 큐레이션 · 풀 게임화 · 동적 추천/길찾기 · (S7 결정에 따른) 댓글·공유. 신규 기능은 가설 검증 통과 → 대표 보고 → 백로그 편입 경로로만.
