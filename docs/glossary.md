## ID 체계
- 화면: `A0`, `S1~S12`(캐논 고정), 신규는 S13+ 등록 후 사용
- 기능: `HTM-[화면]-[##]` (예: HTM-S3-01)
- UI 요소: `BTN_/INP_/LST_/CRD_/SHT_/NAV_/CHIP_` + 화면 + 순번 (예: BTN_S2_01)
- 정책: `POL-U-##`(유저) / `POL-S-##`(서비스) · 오픈이슈: `O#` · 이벤트: snake_case(`record_done`)
